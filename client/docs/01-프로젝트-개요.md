# 프로젝트 개요

## 1. 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | SpringTutorial Dashboard |
| 설명 | 개인 대시보드 웹 애플리케이션 |
| 프론트엔드 | React + TypeScript + Vite |
| 백엔드 | Spring Boot + MyBatis + H2 DB |
| 인증 방식 | JWT (HttpOnly Cookie) |

## 2. 기술 스택

### 2.1 프론트엔드

| 기술 | 버전 | 용도 |
|------|------|------|
| React | 18.x | UI 프레임워크 |
| TypeScript | 5.x | 타입 안전성 |
| Vite | 5.x | 빌드 도구 |
| React Router | 6.x | 라우팅 |
| Axios | 1.x | HTTP 클라이언트 |
| React Query | 5.x | 서버 상태 관리 |
| React Grid Layout | 1.x | 대시보드 레이아웃 |
| Cesium | 1.x | 3D 지도 |
| Three.js | - | 3D 렌더링 |
| Tailwind CSS | 3.x | 스타일링 |

### 2.2 백엔드

| 기술 | 버전 | 용도 |
|------|------|------|
| Spring Boot | 3.x | 웹 프레임워크 |
| Spring Security | 6.x | 인증/인가 |
| MyBatis | 3.x | ORM |
| H2 Database | 2.x | 임베디드 DB |
| JWT (jjwt) | 0.12.x | 토큰 기반 인증 |
| Lombok | - | 코드 간소화 |
| Swagger/OpenAPI | 3.x | API 문서화 |

## 3. 프로젝트 구조

│   │

## 4. 주요 기능

| 기능 | 설명 |
|------|------|
| 로그인/로그아웃 | JWT 기반 인증, 다중 기기 지원 |
| 기기 관리 | 접속 중인 기기 조회, 강제 로그아웃 |
| 대시보드 | 위젯 기반 개인 대시보드 |
| 날씨 위젯 | OpenMeteo API 연동 날씨 정보 |
| 메모 위젯 | 개인 메모 저장/관리 |
| 채팅 위젯 | 실시간 채팅 (WebSocket) |
| 3D 지도 | Cesium 기반 3D 지도 |
| 로봇 팔 | Three.js 기반 3D 로봇 팔 시뮬레이션 |

## 5. 실행 방법

### 5.1 백엔드 실행

```bash
cd server
./gradlew bootRun
```

### 5.2 프론트엔드 실행

```bash
cd client
npm install
npm run dev
```

### 5.3 접속

- 프론트엔드: http://localhost:5173
- 백엔드: http://localhost:8080
- 기본 계정: admin / 1234
