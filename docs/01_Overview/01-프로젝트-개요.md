# SpringTutorial 프로젝트 개요

## 1. 프로젝트 목적 및 소개
SpringTutorial Dashboard는 사용자가 로그인하여 날씨, 메모, 실시간 채팅, 3D 지도(Cesium), 3D 렌더링(Three.js) 등 다양한 위젯을 관리할 수 있는 **개인화 대시보드 웹 애플리케이션**입니다. 다중 기기 동시 접속 관리와 실시간 양방향 통신을 지원합니다.

## 2. 기술 스택

### 2.1 프론트엔드 (Client)
* **Core**: React 18, TypeScript 5, Vite 5
* **Routing & State**: React Router 6, React Query 5, Context API
* **Network & Comm**: Axios (인터셉터 기반 JWT 갱신), WebSocket (STOMP/SockJS)
* **UI/UX**: Tailwind CSS, React Grid Layout (대시보드), SweetAlert2
* **3D Visualization**: Cesium (지리 공간), Three.js (로봇 팔 시뮬레이션)

### 2.2 백엔드 (Server)
* **Core**: Spring Boot 3.x, Java 17+
* **Security & Auth**: Spring Security 6, jjwt 0.12 (JWT 토큰)
* **Database & ORM**: H2 Database (임베디드), MyBatis 3.x
* **Real-time**: Spring WebSocket
* **Build Tool**: Gradle

## 3. 핵심 시스템 기능
1. **강력한 인증 시스템 (Dual-Cookie & Device Tracking)**
   - 단기 `accessToken` (Session 쿠키)과 장기 `deviceId` (Max-Age 1년 쿠키)를 병행 사용.
   - 브라우저를 닫아도 `deviceId`를 통해 기존 세션을 복구하여 '유령 기기' 현상을 원천 차단.
2. **다중 기기 접속 관리**
   - 사용자별 접속 중인 기기 목록, IP, 브라우저 환경을 식별.
   - 특정 기기 강제 로그아웃(WebSocket 연동) 지원.
3. **위젯 기반 개인화 대시보드**
   - 날씨(OpenMeteo 연동), 메모, 채팅 등의 위젯을 사용자가 자유롭게 배치 가능.
4. **실시간 양방향 통신 (WebSocket)**
   - 다른 기기에서 로그인 시 실시간 알림, 전체 채팅 메시지 실시간 브로드캐스팅.

## 4. 실행 방법

### 백엔드 실행
```bash
cd server
./gradlew bootRun
```

### 프론트엔드 실행
```bash
cd client
npm install
npm run dev
```

### 접속 정보
- 프론트엔드 URL: `http://localhost:5173`
- 백엔드 API URL: `http://localhost:8080`
- H2 데이터베이스 콘솔: `http://localhost:8080/h2-console`
- 기본 관리자 계정: `admin` / `1234`
